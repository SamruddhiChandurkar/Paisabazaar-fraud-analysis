# EDA Visualization Charts

This folder contains selected visualization charts created during the Exploratory Data Analysis (EDA) phase of the PaisaBazaar Fraud Analysis project.

## Included Visualizations

### Univariate Analysis
- Histogram — Age Distribution

### Bivariate Analysis
- Box Plot — Income vs Credit Score

### Multivariate Analysis
- Correlation Heatmap
- Pair Plot
- Count Plot — Credit Score vs Payment Behavior

## Note
More than 20 charts and visualizations were created during the analysis process.  
Only a few representative charts from each analysis category were uploaded to provide a concise overview of the project visualizations.
